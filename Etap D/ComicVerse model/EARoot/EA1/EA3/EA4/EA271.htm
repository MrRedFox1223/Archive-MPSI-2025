Scenario Type
Basic Path Basic Path
Details:
Steps:
1. Administrator loguje się do panelu administratora.<br/>
2. System weryfikuj logowanie.<br/>
3. Administrator otwiera panel administracyjnyc sklepu.<br/>
4. Administrator wybiera opcję "Zmodyfikuj produkt".<br/>
5. System wyświetla listę dostępnych produktów.<br/>
6. Administrator wybiera produkt.<br/>
7. System wyświetla formularz z danymi o wybranym produkcie.<br/>
8. Administrator zmienia wartości wybranych atrybutów produktu.<br/>
9. Administrator klika opcję "Zapisz zmiany"<br/>
10. System zapisuje zmiany w bazie danych sklepu.<br/>
11. System wyświetla komunikat o modyfikajci produktu.<br/>
Exceptions:
2a. Błędne dane logowania (at 1)
4a. Administrator wybrał opcję "Dodaj produkt" (at End)
4b. Administrator wybrał opcję "Usuń produkt" (at End)
Błędne dane logowania Alternate
Details:
Steps:
1. System wraca do widoku logowania i wyświetla komunikat, jakie dane są błędne.<br/>
Administrator wybrał opcję "Dodaj produkt" Alternate
Details:
Steps:
1. System wyświetla formularz na dane nowego produktu.<br/>
2. Administrator wprowadza dane nowego produktu i akceptuje.<br/>
3. System dodaje nowy produkt do bazy danych sklepu.<br/>
4. System wyświetla potwierdzenie dodania nowego produktu do sklepu.<br/>
Administrator wybrał opcję "Usuń produkt" Alternate
Details:
Steps:
1. System wyświetla listę produktów w sklepie.<br/>
2. Administrator wybiera z listy produkt do usunięcia i akceptuje.<br/>
3. System usuwa wybrany produkt z bazy danych sklepu.<br/>
4. System wyświetla komunikat o usunięciu wybranego z oferty sklepu.<br/>
